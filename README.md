# DS4420-Project
### Alexander Tu, Dhillon Murphy, Dylan Weinmann

## The Goal
Our goal is to use machine learning techniques to assess MLB pitchers and provide recommendations to increase their effectiveness, both in what they throw and how they throw it. Each model project will use publicly available data from Statcast.com.

## The Methods
We intend to use a Bayesian model to predict hit probability for a given pitch based on the horizontal and vertical locations of a pitch. This will serve to give advice on where the best locations may be for different pitches. Our second model will be a CNN that uses broadcast images from after a pitcher's release combined with pitch type to predict the likelihood of a batter swinging before the ball has reached the plate. This can be used to identify which pitches are working best for drawing out swings from batters. The third model we will use will be a collaborative filtering model that analyzes a pitcher's arsenal and generates comparisons to similar pitchers, and then provides recommended changes based on how similar pitchers have changed.
