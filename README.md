# DS4420-Project
### Alexander Tu, Dhillon Murphy, Dylan Weinmann

## The Goal
We plan to use machine learning methods to analyze baseball pitching data. This analysis will focus on generating recommendations for pitchers in all facets of pitching, whether it be sequencing, arsenal, or location.

## The Methods
We intend to use a Bayesian model to predict pitch type and results using pitch level data provided by Statcast. Our second model will be a CNN that uses broadcast data to anticipate pitch types and locations prior to pitch release, and analyzing how model accuracy improves over the couse of the pitch. The third model we will use will be a collaborative filtering model that analyzes a pitcher's arsenal and generates comparisons to similar pitchers, which can then be used to produce recommendations for updates to a pitcher's arsenal.
